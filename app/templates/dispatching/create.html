Creating a Dispatch

A dispatch represents a work order or assignment. When you create a dispatch:

  • A unique dispatch number is automatically generated
  • An initial event is created to track the dispatch
  • If an asset is assigned, appropriate detail tables are created automatically
  • The dispatch status starts as "Created"
Priority Levels
  • Low: Non-urgent tasks
  • Normal: Standard priority tasks
  • High: Important tasks requiring attention
  • Critical: Urgent tasks requiring immediate attention
