Editing Model Additional Scheduled Task Plans

Changes to this plan will affect all future maintenance scheduling for assets of this specific make/model combination.

Intervals

Set either meter intervals (in hours) or time intervals (in days). These intervals are in addition to the base asset type plan intervals.

Active Status

Only active plans are used for scheduling maintenance. Inactive plans are preserved for historical reference.

Impact

This model plan supplements the base asset type plan. Assets of this make/model will receive both:

  • Base maintenance from asset type plans
  • Additional maintenance from this model plan
Related Plans
