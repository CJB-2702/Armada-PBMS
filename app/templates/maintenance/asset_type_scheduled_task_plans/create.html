Asset Type Scheduled Task Plans

These plans define maintenance schedules for specific asset types. They specify when maintenance should be performed based on time intervals or meter readings.

Required Fields
  • Plan Name: A descriptive name for the plan
  • Asset Type: The type of asset this plan applies to
  • Action Set: The template action set to use for maintenance
Intervals

Set either meter intervals (in hours) or time intervals (in days) to determine when maintenance should be scheduled.
